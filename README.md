# Cognitive Load Classification from EEG Signals

This project explores the use of EEG sensor data to classify cognitive load
levels using machine learning techniques.

## Objectives
- Understand EEG signal structure and labels
- Extract meaningful time and frequency domain features
- Train ML models to classify cognitive load

## Tools
Python, NumPy, Pandas, SciPy, scikit-learn, MNE

## Status
Dataset exploration in progress
